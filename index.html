Projects

teacherHUB

My front-end capstone project was a single page web application called teacherHUB. It is an all in one resource for teachers. It incorporates a grade book, discipline form, allows for streamlined parent and student communication. It was built using AngularJS, Materialize, and Firebase. This project gave me a strong grasp of asynchronous programming using all CRUD operations.

Link to teacherHUB
teacherHUB Welcome Screen

After a user logs in, this is where they are redirected. From here they can quickly access all features of teacherHUB.

User's Class Select Screen

From this screen a user can create a new class, or select one of their existing classes.

Class Dashboard

Here is the class dashboard, from here the teacher can add, view, and update grades for assignments. They can also add students and view overall student grades.

Student Classroom Details

If a teacher wants more data on how a student is doing in a particular class, they can come to this screen to view all assignments with student grades, and a chart displaying the each student grade compared to the class average.

Student's Individual Assignment Report

Here a teacher can get specific data on a student's assignment and view grade data compared to the class average.

Student Profile

Here a teacher can view a student's profile where they can view: class averages, student discipline reports, link to email parents and/or the student.

Incident Report Form

If a teacher has an issue with a student in class, they can come to this screen to add a discipline report and attach it to the student's profile.

Completed Incident Report

Once the user submits the incident report they are sent to this screen where they have the option to email it to the student's parents and copies it to the administration.

Student Search

Here the user can search for students to view their profiles.
